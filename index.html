Frequently Asked Questions

How do I use the script copy feature?

Simply hover over any script card and click the copy button that appears. The script will be automatically copied to your clipboard for easy use.

Are the scripts safe to use?

All scripts undergo thorough review and testing before being added to our platform. We maintain strict security standards to ensure user safety.

Can I contribute my own scripts?

Yes! We welcome community contributions. Join our Discord server to learn about the submission process and guidelines.

How do I switch between themes?

Use the theme toggle button in the top-right corner to switch between light and dark modes. Your preference will be saved automatically.

Is Eclipse Hub mobile-friendly?

Eclipse Hub is fully responsive and optimized for all devices, from desktop computers to smartphones.
